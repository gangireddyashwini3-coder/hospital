# Hospital Management System

## Project Description
The Hospital Management System is a web-based application designed to manage hospital operations efficiently. It helps in handling patient information, doctor details, staff records, room allocation, medical services, and billing in a centralized system.

## Features
- Admin login and dashboard
- Staff and doctor management
- Patient registration (In-patient and Out-patient)
- Room allocation for admitted patients
- Medical information management
- Billing and payment management
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

├── index.html # Home page
└── user.html # User page

## Installation
1. Install XAMPP.
2. Copy the project folder to `htdocs`.
3. Import the database from the `database` folder into phpMyAdmin.
4. Start Apache and MySQL from XAMPP.
5. Open the project in browser:
6. http://localhost/hospital
